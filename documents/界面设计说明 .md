# 汽车仪表盘系统 - 界面设计说明

> 本文档严格依据《汽车仪表盘系统需求文档》编写，所有颜色值、尺寸、布局、字体均以需求文档为准。

---

## 一、设计概述

### 1.1 设计目标

汽车仪表盘系统采用深色科技风格，以深蓝黑渐变为背景，亮绿色为常规状态色，红色为危险/报警色，营造现代汽车仪表的科技感。整体设计追求信息层级清晰、数据读取直观、报警醒目。

### 1.2 设计原则

- **需求驱动**：所有视觉参数（颜色、字体、尺寸、布局）严格遵循需求文档规定，不自行发挥
- **信息层级分明**：速度表和转速表为主仪表，占据左右两侧最大面积；油量表、水温表为辅助仪表，居中排列；报警灯和转向灯居中排列
- **色彩语义统一**：绿色=正常，红色=危险/报警，灰色=辅助信息
- **资源路径规范**：所有资源通过Qt资源系统（.qrc）管理，禁止使用绝对路径；原始文件名含空格或大小写混用的，统一重命名为全小写无空格格式

## 三、布局结构

### 3.1 整体布局（三列式）

需求文档 2.2.2 明确规定了三列布局：

```
┌─────────────────────────────────────────────────────────────┐
│  菜单栏: [信号控制]  [报警灯测试]  [音效控制]                  │
├───────────────┬───────────────────────┬─────────────────────┤
│               │     [转向灯] ← →      │                     │
│               │                       │                     │
│    速度表      │    油量表   水温表     │      转速表         │
│  Speedometer  │  FuelGauge TempGauge  │    RpmGauge        │
│   左侧固定宽度  │                       │    右侧固定宽度      │
│               │     [报警灯组]         │                     │
│               │   WarningLights       │                     │
└───────────────┴───────────────────────┴─────────────────────┘
```

### 3.2 布局详解

| 区域 | 位置 | 控件 | 说明 |
|------|------|------|------|
| 左列 | 左侧固定宽度 | SpeedometerWidget | 速度表，占据整个左列 |
| 中列 | 中间区域 | TurnIndicatorWidget + FuelGaugeWidget + TempGaugeWidget + WarningLightsWidget | 从上到下：转向灯、油量表/水温表并排、报警灯组 |
| 右列 | 右侧固定宽度 | RpmGaugeWidget | 转速表，占据整个右列 |

### 3.3 推荐尺寸分配

在 900×650 窗口内（扣除菜单栏约25px，可用区域约 900×625）：

| 控件 | 建议尺寸 | 说明 |
|------|----------|------|
| 速度表 SpeedometerWidget | 300×625 | 左列，纵向占满 |
| 转速表 RpmGaugeWidget | 300×625 | 右列，纵向占满 |
| 转向灯 TurnIndicatorWidget | 300×60 | 中列顶部 |
| 油量表 FuelGaugeWidget | 145×200 | 中列中部左 |
| 水温表 TempGaugeWidget | 145×200 | 中列中部右 |
| 报警灯组 WarningLightsWidget | 300×80 | 中列底部 |

## 六、各组件视觉规格

### 6.1 速度表 (SpeedometerWidget)

| 属性 | 值 | 来源 |
|------|-----|------|
| 显示范围 | 0 ~ 240 km/h | 需求 2.2.3 |

| 高速区 | ≥120 km/h | 需求 2.2.3 |
| 高速刻度颜色 | #ff4400（橙色红） | 需求 2.2.3 |
| 指针颜色 | #ff2200（红色） | 需求 2.2.3 |
| 数字颜色 | #00ff88（亮绿色） | 需求 2.2.3 |
| 数字字体 | Arial, 36pt, Bold | 需求 2.2.3 |
| 单位颜色 | #556677（暗灰色） | 需求 4.1 |
| 单位字体 | Arial, 12pt | 需求 4.2 |
| 表盘背景 | #1a1a2e → #0d0d1a 渐变 | 需求 4.1 |
| 数字区背景 | #1e1e32 → #1a1a2e 渐变 | 需求 4.1 |
| 动画 | 平滑插值过渡，系数0.1 | 需求 2.2.3 |

**绘制要点**：
- 表盘使用 `QRadialGradient` 实现从 #1a1a2e（中心）到 #0d0d1a（边缘）的渐变
- 数字显示区使用 `QRadialGradient` 实现从 #1e1e32 到 #1a1a2e 的渐变
- 120 km/h 及以上的刻度和刻度数字使用 #ff4400
- 120 km/h 以下的刻度和刻度数字使用 #00ffaa
- 指针使用 #ff2200，可加轻微发光效果（`QRadialGradient` 或阴影）
- 当前速度值大号显示在表盘中心偏下位置，单位 "km/h" 在数值下方

## 七、菜单栏设计

### 7.1 菜单结构

需求文档 2.2.10 和 2.2.9 规定了三个菜单：

```
[信号控制]          [报警灯测试]           [音效控制]
├── 左转            ├── ☐ ENG 超速报警     ├── 🔊 音量: ████████░░ 80%
├── 右转            ├── ☐ OIL 机油         ├── ☐ 静音
├── 双闪警告        ├── ☐ BAT 电瓶         ├── ──────────
└── 关闭转向        ├── ☐ TEMP 水温        ├── 🔊 引擎音效 测试
                    ├── ☐ FUEL 油量        ├── 🔊 机油音效 测试
                    ├── ☐ BRAKE 刹车       ├── 🔊 电瓶音效 测试
                    ├── ☐ AIR 安全气囊     ├── 🔊 水温音效 测试
                    └── ☐ ABS              ├── 🔊 油量音效 测试
                                          ├── 🔊 刹车音效 测试
                                          ├── 🔊 安全气囊音效 测试
                                          └── 🔊 ABS音效 测试
```

### 7.2 信号控制菜单功能

| 菜单项 | 发送指令 | 说明 |
|--------|----------|------|
| 左转 | `turnSignal:1` | 设置左转向灯 |
| 右转 | `turnSignal:2` | 设置右转向灯 |
| 双闪警告 | `hazard:1` / `hazard:0` | 切换双闪状态 |
| 关闭转向 | `turnSignal:0` | 关闭所有转向灯 |

> 互斥逻辑：选择左转时自动取消右转，反之亦然。

### 7.3 报警灯测试菜单功能

- 8个可勾选菜单项，对应8个报警灯
- 点击可手动开启/关闭对应报警灯
- 用于测试和演示

### 7.4 音效控制菜单功能

| 功能 | 说明 | 来源 |
|------|------|------|
| 音量调节 | 范围 0%~100%，默认 80% | 需求 2.2.9 |
| 静音 | 勾选后禁用所有音效 | 需求 2.2.9 |
| 单项音效测试 | 8个测试项，分别播放对应音效 | 需求 2.2.9 |
